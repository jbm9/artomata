Artomata is a small python program to make big GIF files of Elementary
Cellular Automata. 

For more on Elementary Cellular Automata, see 
http://mathworld.wolfram.com/ElementaryCellularAutomaton.html

This program is very bare-bones, which means that it probably won't do
what you want it to, but it should be simple to build on to meet your
needs.

That's about it.
